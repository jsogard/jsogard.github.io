Whiteboard With Friends

This project started off as me and my roommate's final for our first ever web development class. We were initially tasked with creating a web-app that utilized AngularJS with a PHP backend and a school-provided SQL database. With these stipulations we came up with a social networking web-app in which any user can log on with a unique username and draw on a universal canvas that is updated in real time! The project was a resounding success and we submitted it for an A, but we wanted to keep making it better. We eventually turned it into a site where anybody can create their own whiteboards with special permissions on them for their friends.

This project is now dead because the servers with our data on them have since been wiped and given to another class, but I am now rebuilding it in a solo effort using Python Flaskr .
